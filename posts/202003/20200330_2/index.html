hugoをgithub.io にホストできなくなったとき

久しぶりに記事を記事を書いたら github.io にホストできなくなった．

私の運用はhugo公式のStep-by-step Instructionsと同じ．

解決方法

以下の手順で一度 public フォルダを削除したのち，再度 submodule 登録すればOK．

publicフォルダを削除

git rm -rf --cached public

git submodule で再度登録

git submodule add -b master https://github.com/<USERNAME>/<USERNAME>.github.io.git public
